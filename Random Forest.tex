    \begin{quote}
This example has as objective build a Random Forest Regression. Random
Forest is a collection of decision trees whose results are aggregated
into one final result and can be used as a predictor or a classifier. It
is a powerful models that limits overfitting without substantially
increasing error due to bias. The example below uses a Systolic Blood
Pressure dataset, choosing as input the age and weight of patients and
using the systolic blood pressure as output.
\end{quote}
